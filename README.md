## Description

A property rental application built using nest js, typeorm.

## API Documentation

Postman collection for the api are attached in `docs/Property Rental.postman_collection.json`.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
